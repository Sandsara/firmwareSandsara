Detailed Description

[BLE Command] This class is used to create and object to be called when you write to the generalCalibraring characteristic.

Sandara can perform a calibration of the robot (finding the robot's home position) every time a path finishes in the center position. to activate or deactivate this feature you need to use this characteristic

Properties:

  • Write
  • Read

UUID: 7b204f84-30c3-11eb-adc1-0242ac120002

Usage:

you can activate or deactivate the calibration between programs writing to this characteristic and in the same way you can know if it is activated or deactivated.

Range:

1 to activate calibration between paths. 0 to deactivate calibration between paths.

Note
the calibrations need to make a little movement of the ball so if you activate the calibration you will see that the ball makes a little path in the center of sandara every time a path ends in the center.
