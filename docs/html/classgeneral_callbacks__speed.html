Detailed Description

[BLE Command] This class is used to create an object to be called when you write to the generalSleep characteristic.

Sandsara has 3 movement modes, play, pause, and sleep modes. You can enter Sleep mode using this characteristic. Sleep mode means that Sandsara will stop and go to the center or edge position and turn off the led strip. this means that the current path will be rewind and when you resume sandsara the path will start from the beginning.

Properties:

  • Write

UUID: 7b204a3e-30c3-11eb-adc1-0242ac120002

Usage:

if you want to enter Sandsara in sleep mode just write to this characteristic any character.

Range:

whatever character will be understood as if you want to sleep Sandsara.

Response:

you can receive one of these responses after you write to this characteristic.

  • "ok" means Sandsara is in sleep mode.
Note
all the responses have to be read from the generalErrorMsg characteristic after being notified by it.
