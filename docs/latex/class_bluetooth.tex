revisa si hay algo dispobible por bluetooth. 

Puede recibir e interpretar los siguientes mensajes.
\begin{DoxyItemize}
\item code01, significa que va a transferir un archivo.
\item code02, significa que se esta solicitando el cambio de playlist.
\item code03, significa que se esta solicitando el cambio de orden de reproduccion.
\item code04, significa que se esta solicitando el cambio de paleta de leds.
\item code05, significa que se solicita el cambio de velocidad de motores.
\item code06, significa que se solicita el cambio de velocidad de los leds.
\item code07, significa que se desea cambiar el nombre del bluetooth.
\item code08, significa que se desea entrar en modo de suspencion.
\item code09, significa que se desea pausar el path.
\item code10, significa que se desea salir del modo suspencion o pausa.
\item code11, significa que se desea conocer la version del firmware
\item code12, significa que se desea conocer los parametros guardados en R\+OM.
\item code13, significa que se desea conocer el nombre del path actual.
\item code14, significa que se desea conocer el nombre del siguiente path.
\item code15, significa que se desea conocer la lista de reproduccion actual.
\item code16, significa que se desea cambiar de path por posicion en la playlist.
\item code17, significa que se desea cambiar de path por nombre.
\item code18, significa que se modifico la paleta de colores custom.
\item code19, means that you want to change index\+Increment Variable.
\item code20, means that you want to change the intermediate\+Calibration Variable.
\item code21, means that you want to rewind the playlist.
\item code22, means that you want to change the direction of leds.
\item code23, means that you want to delete a file from SD card.
\item code66, actualizar firmware
\item code80, Reiniciar Sandsara
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
un codigo de reporte que indica lo siguiente
\begin{DoxyItemize}
\item 1, se escribio un nuevo archivo.
\item 0, no hay informacion disponible del bluetooth.
\item 10, se solicito un cambio de playlist, para recuperar el nombre llamar a la funcion miembro \mbox{\hyperlink{class_bluetooth_aac9a318d646a6d71157f2bf1531ee547}{get\+Playlist()}}.
\item 20, se solicito un cambio en order\+Mode, para recuperar el numero llamar a la funcion miembro \mbox{\hyperlink{class_bluetooth_a2f3d9574420a559dca6265cd1632d027}{get\+Order\+Mode()}}.
\item 30, se solicito un cambio en led\+Mode, para recueperar el numero llamar a la funcion miembro \mbox{\hyperlink{class_bluetooth_a166a311de363e2f38abd6e0262259446}{get\+Led\+Mode()}}.
\item 50, se solicita el cambio de velocidad de motores, la velocidad se almacena en la variable miembro speed. speed se obtiene con \mbox{\hyperlink{class_bluetooth_a4eca97c45edd1104f513b67d7715e08b}{get\+Speed()}}.
\item 60, se solicita el cambio de velocidad de los leds, la velocidad se almacena en la variable miembro period\+Led. period\+Led se obtiene con \mbox{\hyperlink{class_bluetooth_a2d8ad3cc0ca47d1581852a171aa8fdc1}{get\+Period\+Led()}}.
\item 70, se solicita cambio de nombre de bluetooth.
\item 80, se solicita entrar en modo de suspencion.
\item 90, se solicita entrar en modo de pausa.
\item 100, se solicita salir del modo suspencion o pausa.
\item 110, se envio la version del firmware por bluetooth.
\item 120, se enviaron datos de sandsara
\item 130, se solicita el nombre del programa actual.
\item 140, se solicita el nombre del programa siguiente.
\item 150, se solicita la lista de reproduccion completa.
\item 160, se solicita cambiar de pista por medio de su posicion en la playlist, para conocer la posicion a la que se desea cambiar llamar a la funcion miembro \mbox{\hyperlink{class_bluetooth_a3ca42731eabc84fd677dae2671b8944e}{get\+Position\+List()}}.
\item 170, se solicita reproducir un archivo, para conocer el nombre llamar la funcion miembro \mbox{\hyperlink{class_bluetooth_a8f9b2c66d448c51908860f63ad75da38}{get\+Program()}}.
\item 180, se modifico la paleta custom en rom.
\item 200, intermediate\+Calibration variable was midified in R\+OM.
\item -\/970, se solicita un reset de fabrica.
\item -\/1, no se reconoce el comando enviado.
\item -\/2, se quiso enviar un numero de bytes incorrecto.
\item -\/3, se excedio el tiempo de respuesta del transmisor en la funcion read\+Line (depende de la variable time\+Out\+Bt, medida en milisegundos)
\item -\/4, se excedio el tiempo de respuesta del transmisor en la funcion read\+Bt (depende de la variable time\+Out\+Bt, medida en milisegundos)
\item -\/5, se intento enviar un archivo con un nombre ya existente.
\item -\/7, no coincide checksum.
\item -\/8, no se pudo crear el archivo.
\item -\/9, se han leido mas de X numero de caracteres sin encontrar un \textquotesingle{}~\newline
\textquotesingle{} en la funcion \mbox{\hyperlink{class_bluetooth_a7ea03128eedbb3edcbebdf9b0f15395e}{read\+Line()}}.
\item -\/11, la playlist que se desea cambiar no existe.
\item -\/21, order\+Mode no valido.
\item -\/31, led\+Mode no valido.
\item -\/51, es un directorio
\item -\/52, el archivo esta vacio
\item -\/53, el archivo no se pudo abrir.\+02
\item -\/54, no se pudo finalizar la actualizacion
\item -\/55, no hay suficiente espacio para el O\+TA.
\item -\/56, Ocurrio un error al actualizar el firmware
\item -\/60, Velocidad no permitida para los motores
\item -\/70, periodo no permitido para los leds
\item -\/80, nombre muy largo para bluetooth
\item -\/81, No se pudo cambiar el nombre del bluetooth, pero se vera el cambio cuando se reinicie.
\item -\/97, se intento un reset de fabrica pero no se confirmo.
\item -\/171, se desea correr un programa que no existe en la SD.
\item -\/172, se intento cambiar a un archivo con una extension no valida.
\item -\/181, numero de colores en una paleta incorrecto.
\item -\/182, numero de posiciones distinto a numero de colores en la paleta.
\item -\/183, numero de colores red distinto a numero de colores en la paleta.
\item -\/184, numero de colores green distinto a numero de colores en la paleta.
\item -\/185, numero de colores blue distinto a numero de colores en la paleta.
